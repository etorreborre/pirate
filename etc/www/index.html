pirate

Pirate is a scala library that provides a mechanism for parsing command line arguments and producing usage strings.

quick start

Documentation is currently light on. The best place to start is the demos:

  • io.mth.pirate.demo.PositionalArgsDemo

release

There is not an official release yet. If you want to try it out, the source can be found at github.
